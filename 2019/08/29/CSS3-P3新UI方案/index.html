P3新的UI方案

UI:你所能看到的所有图形都叫UI

一：文本新增样式

（1）opacity：改变透明度（IE9及以上）
img{
opacity:0.4;
filter:alpha(opacity=40); /* 针对 IE8 以及更早的版本 */
}
父元素opacity=0.5，他不能继承，但是能影响到子元素的透明度。（MDN继承性为0）

但当子元素也设置opacity时：
子元素opacity值大于父元素：不变
小于时，则会应用子元素的值。

（2）rgba：与rgb多了一个a
r：红色值。正整数 | 百分数
g：绿色值。正整数 | 百分数
b：蓝色值。正整数 | 百分数
a：Alpha透明度。取值0~1之间。(颜色的透明度)
此色彩模式与RGB相同,RGB定义了颜色的红绿蓝值,在RGB模式上新增了Alpha透明度.

.rgba{
background-color:rgba(0,0,0,.5)
}
（3）两者区别：opacity不可被继承（但可以影响到子元素透明度），rgba不可。
还有opacity可以对文字调整透明度。
=rgba不可。

三：背景透明文字不透明

原理：利用opacity可以对文字设置透明度
而rgba不能的特点，背景使用rgba
具体：文字设置rgb（0,0,0）–不透明
背景设置透明

四：文字阴影(不可继承)

（1）属性：

text-shadow: color x偏移量 Y偏移量 偏移度
a.第一个值为颜色
b.第二个值为X偏移量（px）
c.第三个值为Y偏移量（px）
d.第四个为模糊半径(px):值越大模糊度越大

（2）可以设置多层模糊颜色

下面是两层阴影
text-shadow: blue 10px 10px 10px,red 20px 20px 10px
（3）应用
1)浮雕文字**
主体字体设为白色
将text-shadow的XY偏移量设置小一点（1px）
2)文字模糊
设置当鼠标移入时文字本身为黑色
当鼠标移上去的时候,文字逐渐模糊消失(hover)
h1:hover{
color: rgba(0,0,0,0);
text-shadow: black 0px 0px 200px;
}
这里将rgba(0,0,0,0)。表示接近背景颜色
渐变效果：transition: 1s;渐这个可以设置这个变化时间在1s。

（3)transition属性：可以设置变化的时间

四：应用（图片下面有一个自己的模糊背景）

（移动端开发）

（1）移动端开发：一定要加
<metaname="viewport" content="width=device-width,initial-scale=1.0,user-scalable=no"/>
（2）一共分为三层：
最上层：为图片层
第二层和第三层：是盒子层和背景层
盒子层：background: rgba(0,0,0,0.5);
效果就是黑色50%透明

背景层：
背景图片因为设置绝对定位，提高了层级盖住了img
所以降层级 z-index=-1;()降一个层级
设置图片，no-repeat
使用background-size：设置铺满
然后设置模糊
filter: blur(5px); 数值越大模糊度越高

五：文字描边

（1）他不算是css3中的内容，需要了解，只有在webjit内核才支持，加前缀(用法):-webkit-text-stroke

/其实就是一种四种方向的描边/
（2） -webkit-text-stroke: 4px pink;
