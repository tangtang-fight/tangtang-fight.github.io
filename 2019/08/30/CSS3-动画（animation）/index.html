CSS3-动画（animation）

    • 一：动画（animation）animation的API

    简写属性

    /step(1,end)–end为默认值,1表示一个关键帧周期分为一步
    而且每个周期分配的一秒，对于step函数来说，它是在0%停了一秒，而不是在0%-25%
    关键帧周期:(0%-25%)就是一个关键帧周期    /

    注意：此动画分了4个阶段，在step在0%，25%，50%，75%都停了一秒，导致最后75-100%这个周期无法进行

    animation: move 4s 2s both running 3 alternate steps(1,end);
/*添加关键帧*/

@keyframes move{
    0%{
        transform: translateY(-100px);
    }
    25%{
        transform: translateY(-50px);
    }
    50%{
        transform: translateY(0px);
    }
    75%{
        transform: translateY(50px);
    }
    100%{
        transform: translateY(100px);
    }
}

    实际运动过程

    1：动画的关键帧（animation-name：关键帧名字；）

    切记这里不是要动画的属性名，而是关键帧名字。
    关键帧名字：@keyframes 关键帧名字{}

    2:动画时间（animation-duration：3s；）

    3：动画的功能（animation-timing-function： ；）

    4：动画延迟（animation-delay）

    定义动画开始前等待时间。（动画外属性）

    5：动画执行次数（animation-iteration-content：3；）

    定义动画的执行次数。（只能影响动画内属性）
    无限次：infinite；
    重复的是关键帧：@keyframes{}

    6：动画方向（animation-direction：）

    属性值：①normal（默认值）；②：reverse（反转）；③：alternate（from-to-form-to 呈现s型运动）；④：alternate-reverse（反转s型运动）
    animation-direction: reverse;（反转，从to到from执行）

    反转（reverse）：
    1）他会改变关键帧的执行方向（从to-from）。
    2）他会改变动画其他的属性（如果动画原来是ease-in（慢到快），反转后会变成（快到慢）

    7：动画状态（animation-fill-mode：3；）

    他有三个属性值：①backwords；②forworks；③：both；

    原：原来动画，执行前在初始位置，执行from-to，最后执行动画完成后，回到初始位置。
    ①backwords：将初始位置，移动到from处，开始执行，动画执行完成回到初始位置
    ②animation-fill-mode:forwards ;初始位置–再执行动画–动画结束停在to的位置
    ③both；from位置开始，结束停在to位置（我们理想的动画）
    初始值：none；（与原来一样）

    8：动画的运行和暂停（animation-play-state）

    定义了动画的运行或者暂停
    属性值：①：running（默认值）；②：paused(暂停)；

    二：动画关键帧(@keyframes)

    语法：

    @keyframes move{
    0%{
        transform: translateY(-100px);
    }
    50%{
        transform: translateY(-50px);
    }
    100%{
        transform: translateY(100px);
    }
}

    这里取代了from-to。改为百分比的形式。
    含义：0%-50%-100%，中百分比分割的是时间。
    也就是从0%的-100px到50%的-50px；动画时间占了总运动时间的一半。
